## Job Tracking and Networking

Centralize applications in the dashboard. Set reminders for follow-ups and discover networking events.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Integrate job tracking with your calendar
  const jobs = await fetch('https://api.nxtjob.com/tracker', {
    headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
  });
  console.log(jobs.applications);
  ```
  ```python
  # Python integration example
  import requests
  response = requests.get('https://api.nxtjob.com/tracker',
                         headers={'Authorization': 'Bearer YOUR_TOKEN'})
  print(response.json()['applications'])
  ```
</CodeGroup>

Track metrics like response rates and adjust strategies. Network via AI-matched connections for referrals.
